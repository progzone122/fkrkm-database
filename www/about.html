База данних ФКРКМ (Прототип 0.1)

Простий прототип програми для управління базою даних коледжу за допомогою файлу CSV.

BackEnd програми розроблен на Python, а FrontEnd на JavaScript (HTML / CSS) за допомогою бібліотеки eel.
Також, були використані бібліотеки / фреймворки для JavaScript: jQuery, Bootstrap + Bootstrap icons
